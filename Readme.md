# Recipe App 

## Getting started

This project is a simple recipe app that allows you to add, edit, and delete recipes.

### Technologies

- FastAPI
- React
- React Router
- React Query
- TailwindCSS
- PostgreSQL
- SQLAlchemy
- Alembic


### Requirements

- Python 3.10+
- Node.js 18+
- PostgreSQL 14+

### Installation

1. Clone the repository
2. Install the dependencies

```bash
cd backend
source venv/bin/activate
pip install -r requirements.txt
alembic upgrade head
python main.py
```

```bash
cd frontend
npm install
npm run dev
```
